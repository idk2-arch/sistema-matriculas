Sistema de MatrIculas

Proyecto desarrollado como parte de una práctica académica de programación, simulando un entorno de trabajo colaborativo utilizando Git y GitHub.

-Descripción del Proyecto
Este sistema de matrículas permite la gestión de:

- Estudiantes
- Docentes
- Cursos
- Matrículas
- Calificaciones

Incluye operaciones básicas de creación, lectura y organización de datos, integrando estructuras de datos y consumo de APIs locales.

}Trabajo Colaborativo

El proyecto fue desarrollado utilizando un flujo de trabajo colaborativo simulado mediante Git:

- Uso de **ramas (branches)** para separar el trabajo por colaborador
- Implementación de cambios independientes
- Integración mediante **Pull Requests**
- Consolidación final en la rama `main`
  
- Flujo utilizado:
1. Creación de rama por colaborador
2. Desarrollo de funcionalidades
3. Commit de cambios
4. Push a GitHub
5. Pull Request
6. Merge a `main`
   
 Ramas del Proyecto
- `main` versión principal del sistema
- `colaborador-ui` mejoras de interfaz y diseño

Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- PHP (backend)
- Git y GitHub

estructUra del Sistema

- `/api` servicios backend (PHP)
- `/public`  interfaz del sistema
- `/estructuras` implementación de listas (simple, doble, circular)
- `/config`  conexión a base de datos

 Objetivo Académico

Aplicar conceptos de programación estructurada, estructuras de datos y control de versiones, simulando un entorno real de desarrollo en equipo.

Evidencia de Trabajo

- Commits individuales por funcionalidad
- Uso de ramas para desarrollo paralelo
- Pull Requests documentados
- Merge final en `main`
